# Volunteer Management System (Java)

A console-based Volunteer Management System built using Java and Object-Oriented Programming principles. The project demonstrates core CRUD operations along with search, validation, and efficient in-memory data handling.

---

## Features

- Add new volunteers with unique ID validation  
- View all volunteer records  
- Update volunteer email  
- Delete volunteer by ID  
- Search volunteer by:
  - ID  
  - Name (exact match)  
  - Name prefix search  
- Retrieve multiple volunteers with same name  
- Basic email format validation  
- Count total volunteers in the system  

---

## Concepts Used

- Object-Oriented Programming (OOP)
  - Classes and Objects  
  - Encapsulation  
- Java Collections Framework
  - ArrayList  
- Searching & Filtering algorithms  
- Basic input validation  
- Modular code design
